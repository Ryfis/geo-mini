# Переключатель версий дизайна GeoChat

## Как переключиться между версиями дизайна

В файле `src/components/ChatPage.tsx` есть константа `DESIGN_VERSION`, которая управляет версией дизайна:

```typescript
const DESIGN_VERSION = 'compact' as 'compact' | 'old';
```

### Доступные значения:

- **`'compact'`** - Компактный дизайн (текущий)
  - Меньшие отступы в заголовке (py-1.5 вместо py-3)
  - Поле ввода сообщения с меньшими отступами (py-2 вместо py-3)
  - Небольшой отступ сверху (pt-2.5)

- **`'old'`** - Оригинальный дизайн
  - Большие отступы в заголовке (py-3)
  - Поле ввода сообщения с большими отступами (py-3)
  - Без отступа сверху (pt-0)

## Инструкция по изменению:

1. Откройте файл `src/components/ChatPage.tsx`
2. Найдите строку: `const DESIGN_VERSION = 'compact' as 'compact' | 'old';`
3. Замените `'compact'` на `'old'` для возврата к оригинальному дизайну
4. Сохраните файл и пересоберите проект

## Пример изменения:

```typescript
// Для компактного дизайна (текущий)
const DESIGN_VERSION = 'compact' as 'compact' | 'old';

// Для оригинального дизайна (для возврата)
const DESIGN_VERSION = 'old' as 'compact' | 'old';
```

## Что изменяется в дизайне:

### Шапка чата:
- **Compact:** `py-1.5` (меньше отступов сверху и снизу)
- **Old:** `py-3` (больше отступов)

### Поле ввода сообщения:
- **Compact:** `py-2` (компактное)
- **Old:** `py-3` (оригинальное)

### Контейнер описания поста:
- **Compact:** `py-1.5` (компактный)
- **Old:** `py-3` (оригинальный)

### Предпросмотр фотографий:
- **Compact:** `py-1.5` (компактный)
- **Old:** `py-2` (оригинальный)

### Общий отступ сверху:
- **Compact:** `pt-2.5` (небольшой отступ от верхнего края)
- **Old:** `pt-0` (без отступа)
