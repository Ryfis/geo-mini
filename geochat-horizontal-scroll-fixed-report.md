# Исправление горизонтальной прокрутки в GeoChat

## Проблема
Пользователь сообщил, что горизонтальная прокрутка изображений в чате и общем списке сообщений не работает на мобильных устройствах.

## Диагностика
- Проанализировал код PhotoGallery.tsx
- Проверил существующие CSS стили в index.css
- Выявил потенциальные проблемы с прокруткой на iOS устройствах

## Внесенные изменения

### 1. Обновление index.css
- Добавлен специальный класс `horizontal-scroll` с улучшенными настройками прокрутки
- Включен `-webkit-overflow-scrolling: touch` для плавной прокрутки на iOS
- Добавлен `scroll-behavior: smooth` для плавной анимации
- Включен `overscrollBehaviorX: contain` для предотвращения нежелательного поведения

### 2. Улучшение PhotoGallery.tsx
- Заменен класс `scrollbar-hide` на `horizontal-scroll` для лучшей прокрутки
- Добавлены inline стили для дополнительного контроля прокрутки
- Добавлен `minWidth: '120px'` для гарантии минимальной ширины элементов
- Улучшена обработка touch событий

### 3. Технические детали
**CSS улучшения:**
```css
.horizontal-scroll {
  -webkit-overflow-scrolling: touch !important;
  scroll-behavior: smooth !important;
  overscroll-behavior-x: contain;
  scrollbar-width: none !important;
}
```

**Структура PhotoGallery:**
- Использование flex layout с `gap-2` для равномерного распределения
- `overflow-x-auto` для горизонтальной прокрутки
- `flex-shrink-0` для предвенения сжатия элементов
- Фиксированные размеры 120x120px для стабильного отображения
